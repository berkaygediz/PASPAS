# Code of Conduct

PASPAS is an open-source project with a simple goal: building a useful tool for Windows users. To keep the community healthy and productive, we expect everyone participating to follow a few basic rules.

## Our Standards

**Be respectful.** Treat everyone with respect. We might disagree on technical decisions, but personal attacks, trolling, or insults will not be tolerated.

**Be constructive.** If you find a bug or want a feature, provide clear details and context. Constructive feedback is welcome; complaining without offering solutions is not.

**Keep it professional.** Avoid sexualized language, harassment, or any behavior that would be inappropriate in a professional setting.

## Enforcement

If you encounter unacceptable behavior, please report it by opening a private [GitHub Security Advisory](https://github.com/berkaygediz/PASPAS/security/advisories/new) or contacting the repository owner directly.

Unacceptable behavior may result in being banned from the project's GitHub spaces.
